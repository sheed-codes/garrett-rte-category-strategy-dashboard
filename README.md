# Garrett RTE Category Performance & Distribution Strategy Dashboard

This project demonstrates category management analytics using a synthetic syndicated retail dataset modeled after NielsenIQ / Circana scanner data. The objective is to replicate how a Category Manager evaluates category performance, promotional dependency, distribution opportunities, and SKU productivity to support retail growth strategy.

---

## Overview

This dashboard simulates real-world category review workflows used in consumer packaged goods (CPG) organizations. The analysis focuses on Ready-To-Eat (RTE) popcorn performance and identifies strategic opportunities for distribution expansion, promotional optimization, and assortment planning.

The project was built using Python (Google Colab) for data preparation and Looker Studio for executive-level visualization.
